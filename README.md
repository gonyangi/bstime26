# bstime26 - Firebase Studio 앱

Firebase Studio에서 만든 [학교 특별실 시간표 관리 프로그램]입니다.

## 기능
- 주차별 특별실 이용 학급 확인
- 특별실 사용 시간 선택
- 교담, 스포츠 강사 시간표 확인
- 학급별 기초 시간표 다운로드 가능
