# **App Name**: ClassSync

## Core Features:

- User Authentication: Secure authentication using Firebase Auth to manage user access.
- Realtime Timetable Sync: Real-time synchronization of timetables using Firebase Firestore to ensure all users have the most up-to-date information.
- Timetable Generation Tool: Use an AI-powered tool to assist in generating a timetable for each room and teacher. The tool helps with scheduling.
- Admin Data Import: Functionality to import existing timetable data via CSV upload for quick setup.
- Interactive Timetable UI: Display timetables in an interactive format with selectable rooms, teachers, and classes.
- Conflict Detection: Highlight conflicts between time ranges
- Printable Class Schedules: Generate printer-friendly and PDF exportable versions of class schedules for easy distribution.

## Style Guidelines:

- Primary color: Sky blue (#7DD3FC) for a clean, academic feel.
- Background color: Very light gray (#F9FAFB) for a subtle, unobtrusive backdrop.
- Accent color: Soft teal (#99F6E4) for interactive elements and highlights.
- Body and headline font: 'PT Sans' for a modern, accessible style.
- Code font: 'Source Code Pro' for displaying code snippets.
- Simple, clear icons for easy navigation and understanding of functions.
- Subtle transition animations to indicate changes.
